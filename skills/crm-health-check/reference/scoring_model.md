# 商机健康度评分模型 v2

总分 100 分。A–E 为正向得分（共 90 分），F 为风险修正项（默认 10 分，按风险扣减）。

---

## A. 基础数据完备度（15 分）

| 检查项 | 分值 |
|--------|------|
| 客户名称已填写 | 1 |
| 商机名称已填写 | 1 |
| 行业已填写（客户行业字段） | 2 |
| 预计金额已填写且 > 0 | 3 |
| 预计成交日期已填写 | 2 |
| 商机描述已填写且非空洞（不能只写"客户有需求""推进中"） | 2 |
| 当前阶段已填写 | 1 |
| 来源渠道已填写 | 1 |
| 竞争对手已填写或跟进记录中有明确提及 | 1 |
| 项目范围/产品方向可识别（关联产品或描述中可推断） | 1 |

**注意**：商机描述若只是泛泛一句话，给 1 分；完全空白给 0 分；有实质内容给 2 分。

---

## B. 互动活跃度（20 分）

| 检查项 | 分值 |
|--------|------|
| 最近 7 天内有跟进记录 | 5 |
| 最近 30 天跟进次数达到行业建议频次 | 5 |
| 最近 30 天至少有 1 次推进型跟进（非维护型） | 5 |
| 最近一次跟进明确了下一步动作/时间/责任人 | 5 |

**跟进内容分类**（语义判断）：
- **维护型**（低价值）：节日问候、泛泛寒暄、"客户暂时没回复"、"持续跟进中"
- **信息补充型**（中价值）：发送资料、简单交流需求、补充背景信息
- **推进型**（高价值）：约会议、深度需求访谈、现场调研、方案汇报、报价讲解、决策层沟通、采购/合同流程确认、一期范围确认

**行业建议跟进频次**（默认，各 playbook 可覆盖）：
- 初步接触阶段：30 天内 ≥ 3 次
- 需求确认阶段：30 天内 ≥ 4 次
- 方案/报价阶段：30 天内 ≥ 3 次
- 谈判阶段：30 天内 ≥ 2 次高质量推进

---

## C. 关键角色覆盖度（20 分）

根据**当前标准阶段**选择对应规则，满分 20 分。

**角色类型定义**：
- **项目发起人**：最初提出需求的人
- **使用负责人**：真正会使用系统/推动落地的人
- **决策者**：老板、总经理、厂长、部门负责人
- **影响者**：IT、财务、采购、上级管理者
- **执行接口人**：日常跟进的对接人/窗口人

**初步接触阶段**：

| 条件 | 分值 |
|------|------|
| 至少有 1 个明确联系人 | 8 |
| 已识别联系人角色 | 6 |
| 已识别项目发起人或使用负责人 | 6 |

**需求确认阶段**：

| 条件 | 分值 |
|------|------|
| 有 2 个及以上不同角色联系人 | 5 |
| 已识别使用负责人 | 5 |
| 已识别项目推动人/内部 champion | 5 |
| 已初步识别决策路径 | 5 |

**方案/报价/谈判阶段**：

| 条件 | 分值 |
|------|------|
| 已接触或确认决策者 | 6 |
| 已接触或确认使用负责人 | 4 |
| 已接触或确认 IT/采购/财务影响角色 | 4 |
| 已识别拍板人及签约流程 | 6 |

---

## D. 阶段证据完备度（25 分）

这是核心维度。不是问"停了多久"，而是问"当前阶段该有的证据有没有"。
从跟进记录内容语义提取，判断关键里程碑是否已达成。每项 5 分，共 5 项 = 25 分。

**初步接触**：
1. 已明确客户基本业务场景
2. 已识别客户主要痛点
3. 已明确项目发起人/窗口人
4. 已确认是否有进一步沟通意愿
5. 已约定下一次有效沟通

**需求确认**：
1. 已明确当前问题（非泛需求）
2. 已确认涉及的业务部门
3. 已明确想解决的核心目标
4. 已判断是新建、替换还是扩展
5. 已约深度调研或演示

**调研/诊断**：
1. 已完成现场调研或深度访谈
2. 已形成现状问题清单
3. 已确认一期范围/试点范围
4. 已识别接口/集成约束
5. 已初步判断预算与项目优先级

**方案设计**：
1. 已形成针对性方案（非通用 PPT）
2. 方案已映射客户现状问题
3. 已明确边界与不做范围
4. 已安排方案汇报对象（含管理层）
5. 已确认客户反馈机制

**报价/商务**：
1. 已正式报价
2. 已解释报价逻辑
3. 已识别客户买断/订阅偏好
4. 已识别竞品/替代方案
5. 已明确预算区间或接受度

**谈判/采购**：
1. 已明确拍板人
2. 已明确采购/合同流程
3. 已确认付款条款偏好
4. 已识别法务/招采阻力
5. 已存在签约前置动作清单

**待签约**：
1. 合同主体明确
2. 金额范围基本一致
3. 付款节点已基本确认
4. 实施范围已锁定
5. 预计签约时间有可信依据

---

## E. 行业适配度（10 分）

根据适用 playbook 的最佳实践判断，满分 10 分。

| 检查项 | 分值 |
|--------|------|
| 当前阶段动作符合行业常规成交路径 | 3 |
| 当前跟进重点符合该行业角色重点 | 2 |
| 当前方案范围控制符合行业最佳实践 | 2 |
| 当前价值表达符合行业关注点 | 2 |
| 当前推进节奏合理（未超出行业建议停留周期） | 1 |

---

## F. 风险修正项（10 分，默认满分，按风险扣减）

| 风险项 | 扣分 |
|--------|------|
| 只有单线程联系人（仅有窗口人，无决策链延伸） | -2 |
| 长期无决策层信号（进入方案阶段后仍未接触决策者） | -2 |
| 明显无预算或预算极弱（跟进记录中有负面信号） | -2 |
| 强竞品介入且无差异化策略 | -2 |
| 需求范围明显失控（反复变动，边界不清） | -1 |
| 客户优先级不高/时机不成熟（客户明确说"暂缓"或"等领导回来"超过30天） | -1 |
| 预计成交日期明显失真（多次延期或与实际进度严重不符） | -1 |
| 跟进记录长期无实质推进（近30天全为维护型跟进） | -2 |

风险扣分最低为 0 分（不出现负分）。

除扣分外，输出风险标签：组织风险 / 预算风险 / 竞争风险 / 范围风险 / 时机风险 / 交付风险

## 健康度等级

| 分数 | 等级 | 定义 |
|------|------|------|
| 85–100 | 🟢 健康 | 处于正常推进轨道，可争取提速 |
| 70–84 | 🟡 需关注 | 有明显短板，需修正打法 |
| 50–69 | 🟠 高风险可挽救 | 存在明显阻塞，若 1–2 周内不突破可能滑落 |
| 0–49 | 🔴 高风险 | 已严重偏离正常成交路径，需重估投入策略 |

---

## 七个标准阶段（用于语义映射）

CRM 阶段名称因企业而异，执行时需将实际阶段名语义映射到以下标准阶段之一：

1. **初步接触** — 刚建立联系，了解基本情况
2. **需求确认** — 明确需求方向，确认有推进意愿
3. **调研/诊断** — 深入现场调研，梳理现状问题
4. **方案设计** — 输出针对性解决方案
5. **报价/商务** — 进入价格谈判和商务环节
6. **谈判/采购** — 合同条款、采购流程推进
7. **待签约** — 即将签约，最终确认阶段
